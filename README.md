# React-redux-boilerplate

## What?

A boilerplate made to make creating a react app easier than ever. Just npm install and npm start it. 

## Packages:

contains the following packages:

* react
* react-redux
* react-dom
* react-router-dom
* react-router-redux
* redux
* redux-thunk
* history
* redux-devtools-extension
* bootstrap
* react-bootstrap
